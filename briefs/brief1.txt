Проект - использование подарочных сертификатов в екоммерс энтерпрайз организации.

В проекте уже есть сервисы:
- интернет-магазин
- оффлайн-точки продаж (POS)
- платёжный сервис (отвечает за платежи, Payments)
- учётная система (отвечает за заказы, Orders)
- CRM-система (отвечает за покупателей, Clients)

В проекте уже есть потоки данных:
- заказы All_SalesOrder
- платежи All_Payments
- покупатели All_Clients

Чего пока нет:
- сервиса процессинга подарочных сертификатов
- потоков про подарочные сертификаты (необходимость не ясна)
- интерфейсов UI для покупателей и для внутренних задач
