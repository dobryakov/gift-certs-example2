# AI Decisions

- **Планирование релизов и этапов:** Пользователь делегировал определение сроков и зависимостей («решай сам»). Сформирован поэтапный план задач (см. `specs/tasks.md`) с предложенными сроками и зависимостями.
- **Формат диаграмм:** Выбран текстовый формат Mermaid и BPMN XML для дальнейшего рендеринга, поскольку прямое создание графических файлов недоступно в окружении.
- **HTTP API охват:** Добавлены только диагностические и поддерживающие эндпоинты, поскольку командные операции выполняются через Kafka, что согласуется с корпоративными правилами.
